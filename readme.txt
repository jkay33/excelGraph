This application will search the current working directory for an excel or csv file.
The application shall run 3 tasks:
1. Parse the excel files and store into a dataframe.
2. Export the data to an excel files.
3. Utilizing excel it will graph the data that was parsed.


To run the application:
1. Ensure Python is installed on the machine.
2. Save the folder to the machine and move the excel or csv into the folder.
3. Open command prompt and cd to the folder.
4. Type "pip install -r requirements.txt"
5. After all dependencies are installed, type "python excelGraph.py"

After the script has completed, open the directory and an excel file titled graphBook will be present.
